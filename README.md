# ExploreMath

An interactive math learning platform for German and English pupils (grades 5–12). Learn mathematics through experiments, interactive diagrams, and deep understanding.

## 🧮 What is this?

ExploreMath teaches math the way it should be taught — through exploration and understanding, not memorization. Every lesson is an experiment where you discover mathematical principles yourself.

## 🛠 Tech Stack

- [Astro](https://astro.build) — Static site generation
- [Svelte](https://svelte.dev) — Interactive components
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [D3.js](https://d3js.org) — Mathematical visualizations
- [KaTeX](https://katex.org) — Math rendering
